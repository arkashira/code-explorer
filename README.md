# Code Explorer
A plugin for IDEs to provide code explanations and related components.

## Features
* Real-time hover tooltips show component descriptions
* Contextual navigation links appear in editor sidebar

## Requirements
* Python 3.8+
* pytest for testing

## Installation
1. Install the plugin using your IDE's plugin manager.
2. Configure the plugin to point to your code base.

## Usage
1. Open your code base in your IDE.
2. Hover over a component to see its description and links.
3. Click on a link to navigate to the related component.
